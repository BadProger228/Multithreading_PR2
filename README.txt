Короткий опис проекту:

--------------------------------------------------------------------------------------

Проект поділений на 2 файли, Main і Solve. В основному файлі просто запитується скільки потоків хоче запустити користувач. 
Основна логіка знаходиться в файлі Solve. В файлі реалізований однойменний клас. В нього є приватні поля для максимального та мінімально можливого значення в масиві рандомних чисел.
В конструкторі передається з Main скільки потоків треба відкрити в цій програмі. Розберемо методи в класі
- printArray - виводить красиво масив інтів
- generateArray - генерує масив випадкових інтів випадкової довжини
- start - метод який запускає роботу класу
